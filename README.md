# STEP Face Painter

3D CADの標準フォーマットであるSTEPファイル（.stp / .step）をブラウザ上で読み込み、3Dモデルの「面（Face）」に対して直感的に色を塗ることができるWebアプリケーションです。3D形状の確認や、面ごとの属性・カラー分けに活用できます。

開発にはブラウザ上で動作する強力な3D CADカーネル「OpenCascade.js」が使用されています。

## 主な機能

- **STEPファイルの読み込み**: ローカルにあるSTEPファイルをドラウザにドラッグ＆ドロップまたはファイル選択で高速に読み込み・表示します。
- **フェイス・ペインティング（面への着色）**: 3Dモデルの特定の「面（Face）」を選択し、ブラシやカラーパレットを使って直感的に色を塗ることができます。
- **豊富なカラーツール**:
  - 自由な色を選択できるブラシカラー
  - よく使う色をまとめたカラーパレット
  - 塗ったカラーデータの保存（💾）および読み込み（📥）機能
- **充実した3Dビュー操作**:
  - 軸方向（+X, -X, +Y, -Y, +Z, -Z）への素早い視点切り替え
  - 🎯 センタリング機能（モデルを画面中央に配置）
  - エッジ（輪郭線）の表示/非表示の切り替え
  - グリッド（背景の網目）の表示/非表示の切り替え
  - 便利なUndo（元に戻す）機能
- **詳細なタグ情報（Tag Information）の確認**:
  - 選択した面の「Face ID」の表示
  - サーフェス（Surface）のタイプ情報
  - ポリゴン数（Triangles）の確認
- **スクリーンショット撮影**: 画面内の3Dモデルをきれいにキャプチャして保存できます。

## 使い方

1. **ファイルの読み込み**: 「📂 ファイルを開く」ボタンから、または画面へのドラッグ＆ドロップでSTEPファイルを読み込みます。
2. **色の選択**: カラーパレットまたはブラシカラーから、塗りたい色を選択します。
3. **色を塗る**: 3Dビュー上のモデルの面をクリックして着色します。
4. **視点の変更**: マウス操作（回転・ズーム・パン）や、視点切り替えボタンを使ってモデルを観察します。
5. **データの保存**: 必要に応じて「カラーを保存」から作業内容をエクスポートします。

## 技術スタック / クレジット
